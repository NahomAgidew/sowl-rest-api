# sowl-rest-api [![Build Status](https://travis-ci.org/dwyl/esta.svg?branch=master)](https://travis-ci.org/dwyl/esta)
REST API demo for scholarshipowl.com

## Setup
<code>composer update && composer install</code>

## Dependencies
* PHP 7
* composer
* slim
* Toast
* MySQL

## Testing
